# 개인정보 처리방침 (초안)

> ⚠️ 이 문서는 개발자가 작성한 초안이다. 실제 서비스 공개 전 **법률 검토 1회가 필요하다**
> (아동 이용자가 예상되는 서비스이므로 특히 만14세 미만 동의 절차).

## 1. 수집하지 않는 것

TypingVerse는 다음을 **일절 수집하지 않는다**.

- 이메일 주소
- 실명
- 생년월일
- 전화번호
- 주소
- 결제 정보 (현재 유료 기능 없음)

계정 생성에 회원가입이 없다. 앱을 열면 익명 계정이 만들어진다.

## 2. 수집하는 것

| 항목 | 목적 | 보관 |
|---|---|---|
| 닉네임 | 순위 표시 | 계정 삭제 시까지 |
| 아바타 선택값 | 프로필 표시 | 계정 삭제 시까지 |
| 기기 식별자(해시) | 같은 기기에서 내 기록을 이어보기 위함. **원문은 저장하지 않고 HMAC 해시만 보관** | 계정 삭제 시까지 |
| 연습·게임 기록(속도·정확도·시간) | 통계, 순위, 맞춤 드릴 | 계정 삭제 시까지 |
| 타건 간격(익명 수치 배열) | 순위 부정행위 검증 | 검증 후 90일 |
| 학년대(선택, 자기신고) | 또래 순위 분류 | 계정 삭제 시까지 |

접속 IP는 Cloudflare 인프라에서 보안 목적으로 일시적으로 처리될 수 있으나 서비스가 별도로 저장하지 않는다.

## 3. 아동 이용자에 대하여

- 이메일·생년월일·실명을 받지 않아 수집 항목을 최소화했다.
- 다만 **기기 식별자와 이용 기록도 결합에 따라 개인정보에 해당할 수 있으므로**,
  "동의가 전혀 필요 없다"고 보지 않는다.
- 학교·학급 단위 이용 시에는 교사·학교를 통한 안내와 동의 절차를 함께 운영한다.
- 아동을 대상으로 한 맞춤형 광고를 게재하지 않는다. (현재 광고 자체가 없다)
- 확률형 보상은 무료 재화로만 제공하며 현금 구매 요소가 없다.

## 4. 제3자 제공

제3자에게 제공하지 않는다. 서비스 운영에 사용하는 처리 위탁은 아래와 같다.

| 수탁자 | 목적 |
|---|---|
| Cloudflare | 웹 호스팅, 데이터베이스(D1) |

## 5. 이용자의 권리

- **기록 삭제**: 설정 화면에서 모든 기록을 삭제할 수 있다(기기 저장분 + 서버 계정).
- **기기 이전**: 전학 코드로 다른 기기에 기록을 옮길 수 있다. 코드는 24시간 후 만료되며 1회만 사용된다.
- 문의처: (운영 시작 전 기입)
